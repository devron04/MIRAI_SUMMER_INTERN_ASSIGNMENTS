# 2. Hook

## What is a Hook?

A **hook** is a mechanism that allows a program to run specific code automatically when a particular event occurs.

## Examples

- **Git Hook** – Runs before or after Git actions (e.g., `pre-commit`).
- **React Hook** – Functions like `useState()` and `useEffect()` to manage state and side effects.
- **Event Hook** – Executes code when an event (such as a button click) happens.

## Key Points

- Hooks automate tasks.
- They execute code in response to events.
- Commonly used in Git, React, and many software frameworks.
